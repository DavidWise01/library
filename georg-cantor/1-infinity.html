Infinity Formalized

Set theory, cardinality, the diagonal argument, and the hierarchy of infinities
CANTOR'S DIAGONAL ARGUMENT  (1891)  — animated

I. Set Theory — A New Language for Mathematics

A set is any collection of definite, distinguishable objects considered as a whole. This sentence, so simple it appears trivial, is Cantor's founding definition — and it is revolutionary. Before Cantor, mathematicians worked with numbers, functions, and geometric figures. They did not have a general theory of collections as mathematical objects in their own right. Cantor gave them one.

The objects in a set are called its elements or members. If a is an element of a set A, we write a ∈ A. The collection of all natural numbers {1, 2, 3, ...} is a set. The collection of all even numbers is a set. The collection of all real numbers between 0 and 1 is a set. The collection of all sets that do not contain themselves is — as Russell would discover — not a set in the naive sense, but that discovery came later and required the careful axiomatization of the theory.

The Empty Set

The empty set ∅ is the set that contains no elements. It is unique: there is only one empty set. This may seem uninteresting, but the empty set is the foundation from which, in axiomatic set theory, all mathematical objects are ultimately constructed. The number zero is not a primitive — it is the cardinality of the empty set. The number one is the cardinality of the set {∅}. The number two is the cardinality of {∅, {∅}}. And so on, for all the natural numbers.

Basic Operations

BASIC SET OPERATIONS A ∪ B  =  union  — all elements in A or B (or both)
A ∩ B  =  intersection  — only elements in both A and B
A \ B   =  difference  — elements in A but not in B
℘(A)   =  power set  — the set of all subsets of A
A ⊆ B  =  A is a subset of B  — every element of A is also in B
|A|     =  cardinality of A  — the "number" of elements in A

The power set is the operation that will later produce Cantor's hierarchy of infinities. If A has n elements, then ℘(A) has 2n elements. The power set is always strictly larger than the original set. This holds even when A is infinite — and this is not obvious, and it required Cantor to prove it.

Example

Power Set of a Small Set

Let A = {a, b, c}. Then |A| = 3. The power set ℘(A) contains 2³ = 8 subsets: ∅, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c}. This 2n relationship persists into the infinite.

Key insight

Sets Can Contain Sets

A set can be an element of another set. The power set ℘(A) is a set whose elements are themselves sets. This self-referential property is both powerful and — as Russell showed — dangerous without careful axioms.

Foundation

Everything is a Set

In modern axiomatic set theory (ZFC), every mathematical object — numbers, functions, ordered pairs, geometric spaces — is ultimately defined as a set. Set theory is not a branch of mathematics; it is its foundation.

II. Cardinality — Counting Without Counting

How do you compare the sizes of two infinite sets? You cannot count to the end of an infinite set — there is no end. Cantor's decisive insight was to separate the concept of "size" from the concept of "counting." Two sets have the same cardinality if and only if there is a bijection between them — a one-to-one correspondence that matches every element of the first set with exactly one element of the second, and vice versa.

For finite sets, this is just ordinary equality of count. {a, b, c} and {1, 2, 3} have the same cardinality because you can pair them: a↔1, b↔2, c↔3. No element is left unpaired. This is really what it means for two collections to "have the same number" — the bijection is primary, the number is derived from it.

Galileo's Paradox — Now Formalized

In his Two New Sciences (1638), Galileo observed a puzzle: the natural numbers and the perfect squares seem to be in one-to-one correspondence, even though the squares are a proper subset of the naturals. He thought this showed that comparisons of size break down for infinite collections. Cantor formalized this observation and showed it is not a paradox but a definition:

GALILEO'S PARADOX — RESOLVED Naturals:  1  2  3  4  5  6  7  8  9 ⋯
           ↕  ↕  ↕  ↕  ↕  ↕  ↕  ↕  ↕
Squares:   1  4  9 16 25 36 49 64 81 ⋯

Bijection: n ⟷ n². Every natural maps to exactly one square, and every square
maps back to exactly one natural. Therefore: |ℕ| = |{perfect squares}|.
A proper subset of an infinite set can have the same cardinality. This is
not paradox — it is the definition of infinity.

Cantor made this a definition rather than a paradox: an infinite set is precisely a set that has the same cardinality as some of its proper subsets. This is the formal content of infinity. It is counterintuitive only until you accept that the bijection is the primary concept and "size" is secondary.

"A set is infinite if and only if it has the same cardinality as one of its proper subsets."

— Dedekind's equivalent formulation, later named the Dedekind-infinite definition

III. Countable and Uncountable — Not All Infinities Are Equal

The first great theorem of Cantor's theory of cardinality: there are at least two different sizes of infinity. The natural numbers define the smallest infinite cardinality, denoted ℵ₀ (aleph-null). Any set that can be put into bijection with the natural numbers has cardinality ℵ₀ and is called countably infinite. The question is: are all infinite sets countable?

The Rationals Are Countable

This is surprising. There are infinitely many rational numbers between any two distinct rationals — the rationals are "dense" in a way the integers are not. Yet they are countable. Cantor's proof uses a clever enumeration: arrange all positive fractions p/q in an infinite grid (row p, column q), then traverse the grid diagonally, zigzagging through every fraction in a systematic order, skipping duplicates. This traversal is a bijection between the naturals and the positive rationals.

DIAGONAL ENUMERATION OF THE RATIONALS 1/1 → 1/2   1/3 → 1/4   ⋯
     ↓    ↗      ↓    ↗
2/1   2/2   2/3   2/4   ⋯
↓    ↗          ↗
3/1   3/2   3/3   3/4   ⋯
⋯

Enumerate: 1/1, 1/2, 2/1, 3/1, 2/2, 1/3, 1/4, 2/3 ⋯ (skip duplicates)
This is a bijection ℕ → ℚ⁺. Therefore: |ℚ| = ℵ₀

The Reals Are NOT Countable — The First Proof (1874)

In 1874, Cantor published his first proof that the real numbers cannot be enumerated. The argument used the nested interval theorem: given any proposed enumeration of all real numbers in an interval, you can construct a nested sequence of intervals each contained in the previous, with lengths shrinking to zero, such that the limit point is not in the enumeration. This is a topological argument — powerful but somewhat opaque.

Seventeen years later, Cantor found a cleaner, more elegant proof. It is one of the most beautiful arguments in all of mathematics.

IV. The Diagonal Argument — 1891

The diagonal argument is Cantor's masterpiece. It proves not only that the real numbers are uncountable but provides a general method — a "diagonalization" technique — that would reappear throughout twentieth-century mathematics and logic. The proof is by contradiction.

The Argument, Step by Step

  • 1
    Assume for contradiction that the real numbers between 0 and 1 are countable. Then there exists an enumeration: r₁, r₂, r₃, ... listing every real number in (0,1) exactly once.
  • 2
    Write each number in its decimal expansion. So r₁ = 0.d₁₁ d₁₂ d₁₃ ..., r₂ = 0.d₂₁ d₂₂ d₂₃ ..., and in general rₙ = 0.dₙ₁ dₙ₂ dₙ₃ ..., where dₙₖ is the k-th decimal digit of the n-th number.
  • 3
    Construct a new number d = 0.e₁ e₂ e₃ ... by the rule: eₙ differs from dₙₙ (the n-th digit of the n-th number). For concreteness: if dₙₙ = 4, set eₙ = 5; otherwise set eₙ = 4. (Avoid 0 and 9 to sidestep decimal representation ambiguity.)
  • 4
    Observe that d is a real number between 0 and 1 (it has a valid decimal expansion), yet d differs from r₁ in its first decimal place, from r₂ in its second, from r₃ in its third, and from rₙ in its n-th decimal place for every n.
  • 5
    Contradiction: d is not equal to any rₙ in the list. But we assumed the list contains every real number in (0,1). Therefore d cannot exist — but it clearly does. The assumption that the reals are countable is false.
  • 6
    Conclusion: The real numbers are uncountable. Their cardinality, called the cardinality of the continuum and denoted 𝔌 or 2ℵ₀, is strictly greater than ℵ₀. There are more real numbers than there are natural numbers. Not all infinities are equal.
THE DIAGONAL CONSTRUCTION — FORMAL List (assumed complete):
r₁ = 0. d₁₁ d₁₂ d₁₃ d₁₄ d₁₅ ⋯
r₂ = 0. d₂₁ d₂₂ d₂₃ d₂₄ d₂₅ ⋯
r₃ = 0. d₃₁ d₃₂ d₃₃ d₃₄ d₃₅ ⋯
r₄ = 0. d₄₁ d₄₂ d₄₃ d₄₄ d₄₅ ⋯
r₅ = 0. d₅₁ d₅₂ d₅₃ d₅₄ d₅₅ ⋯
⋯

Constructed number:
d = 0. e₁ e₂ e₃ e₄ e₅ ⋯
where eₙ ≠ dₙₙ for all n.

Therefore: d ≠ rₙ for all n.  d is NOT IN THE LIST. Contradiction. ∎

"The argument is so simple and so general that it is a wonder it was not found earlier. It proves not only that the reals are uncountable, but that for any set S, the power set ℘(S) has strictly greater cardinality than S."

— Modern assessment, widely attributed in textbooks on set theory

Why It Generalizes

The beauty of diagonalization is that it is not really about real numbers at all. It is about the impossibility of listing all elements of a power set. Let S be any set and f: S → ℘(S) any function. Define D = {x ∈ S : x ∉ f(x)}. Then D is a subset of S (so D ∈ ℘(S)), but for every x in S, D ≠ f(x) (because either x ∈ D and x ∉ f(x), or x ∉ D and x ∈ f(x)). Therefore f is not surjective. No function from S to ℘(S) can be surjective. Therefore |℘(S)| > |S|. Always. For every set, finite or infinite.

V. The Aleph Numbers — A Hierarchy of Infinities

Cantor introduced the Hebrew letter aleph (ℵ) to denote his transfinite cardinal numbers. The subscript indicates the position in the hierarchy. This notation was not arbitrary: Cantor, a deeply religious man who corresponded with theologians about the philosophical status of the actual infinite, chose the first letter of the Hebrew alphabet deliberately. He saw the transfinite as a creation of God, a domain that human reason could explore but not exhaust.

Cardinal Meaning Example Sets
ℵ₀ Countable infinity — the smallest infinite cardinal ℕ (naturals), ℤ (integers), ℚ (rationals), algebraic numbers
𝔌 = 2ℵ₀ Cardinality of the continuum ℝ (reals), any interval (a,b), ℝn, all continuous functions
ℵ₁ The next infinite cardinal after ℵ₀ The set of all countable ordinals
ℵ₂ The next after ℵ₁ Defined by the ordinal hierarchy
ℵα The α-th infinite cardinal (for any ordinal α) There are as many alephs as there are ordinals — a proper class

The Continuum Hypothesis

Cantor proved that 𝔌 = 2ℵ₀ (the cardinality of the reals equals 2 raised to aleph-null). He believed, but could not prove, that 𝔌 = ℵ₁ — that the cardinality of the continuum is the very next infinite cardinal after ℵ₀, with no infinity in between. This is the Continuum Hypothesis:

THE CONTINUUM HYPOTHESIS (CH) There is no set whose cardinality is strictly between ℵ₀ and 2ℵ₀.

Equivalently:  2ℵ₀ = ℵ₁

Cantor spent years trying to prove this. He failed. He did not know why.
Hilbert listed it as Problem #1 in his famous 1900 list of 23 open problems.
The resolution came in two parts, decades after Cantor's death:

  1940: Gödel proved CH is consistent with ZFC — you cannot disprove it.
  1963: Cohen proved CH is independent of ZFC — you cannot prove it.

The Continuum Hypothesis is undecidable from the axioms of set theory.
It can be freely assumed either true or false. Neither leads to contradiction.

Cantor's failure to prove the Continuum Hypothesis contributed to his mental breakdowns. He cycled between believing he had proved it and believing he had disproved it, only to find errors in both directions. He could not have known — no one could have known — that the question was fundamentally unanswerable from within the axioms he was working with. He was, in his own way, running into the limits that Gödel would later formalize.

There Is No Largest Infinity

The power set operation provides an inexhaustible supply of ever-larger infinities. Starting from ℵ₀, we form 2ℵ₀, then 22ℵ₀, and so on, each strictly larger than the last. The sequence of aleph numbers ℵ₀, ℵ₁, ℵ₂, ... is itself infinite, and beyond all of them is ℵω (aleph-omega), and beyond that ℵω+1, and so on for ever. There is no ceiling. There is no "set of all cardinals" — that collection is too large to be a set; it is a proper class. The hierarchy of infinities is itself infinite, and transfinitely so.

"In mathematics the art of proposing a question must be held of higher value than solving it."

— Georg Cantor

VI. Cantor's Theorem — The Power Set Is Always Larger

The crown of Cantor's cardinal arithmetic is the theorem that bears his name:

CANTOR'S THEOREM For any set A:    |A| < |℘(A)|

The cardinality of the power set of A is strictly greater than the cardinality of A.
This holds for every set — finite, infinite, or transfinite.

Proof sketch (diagonal argument applied generally):
Suppose f : A → ℘(A) is any function.
Define: D = { x ∈ A : x ∉ f(x) }
Then D ∈ ℘(A), but for any a ∈ A: D ≠ f(a).
(If a ∈ D then a ∉ f(a) so f(a) ≠ D; if a ∉ D then a ∈ f(a) so f(a) ≠ D.)
Therefore f is not surjective. No surjection A → ℘(A) exists.
Therefore |A| < |℘(A)|.  ∎

This theorem has a disquieting corollary: there is no largest set, no universal set, no "set of all sets" (in any set theory that takes Cantor's theorem seriously). If such a set U existed, ℘(U) would have to be a set and would have cardinality greater than U — but U was supposed to contain all sets. Contradiction. The proper class of all sets is not itself a set. This is not a technicality; it is a fundamental fact about the nature of mathematical infinity.

Consequence

No Universal Set

If a set of all sets existed, Cantor's theorem would produce a set larger than it. Therefore no universal set exists. The "universe" of set theory is a proper class, not a set.

Consequence

No Largest Cardinal

For any cardinal κ, the cardinal 2κ is strictly larger. There is no ceiling. The cardinals form a proper class — a collection too large to be a set.

Application

The Reals as Power Set

The reals ℝ can be put in bijection with ℘(ℕ). So |ℝ| = 2ℵ₀. The uncountability of the reals is the uncountability of a power set. Cantor's theorem explains why.
